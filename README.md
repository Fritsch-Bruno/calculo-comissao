# calculo-comissao
Programa em Python para cálculo de comissão, bônus e salário líquido de um vendedor

##  Sobre o projeto
O programa calcula:
- comissão de vendas
- bônus
- salário bruto
- descontos
- salário líquido

##  Tecnologias utilizadas
- Python

##  Como executar

1. Abra o terminal
2. Navegue até a pasta do projeto
3. Execute o arquivo com:

```bash
python CalculoVenda.py
